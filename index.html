채팅 로그 편집기

Roll20 / d20.io 채팅 익스포트를 정리하고, 텍스트를 직접 편집한 뒤 수정된 HTML 파일로 내보냅니다

HTML 편집 (큰 창)
창 모서리를 드래그해 크기를 조절할 수 있습니다. 긴 줄은 자동으로 줄바꿈됩니다.
이 메시지의 HTML 편집
이 메시지에 해당하는 HTML만 수정합니다. 적용하면 미리보기에 바로 반영됩니다. (Esc: 취소)

핸드아웃 삽입
· 테두리 박스 3단(제목/부제/본문) 형식으로 삽입됩니다. 삽입 후 본문 편집 모드에서 직접 수정할 수도 있습니다.

원본 입력 (직접 수정 가능)

변환 규칙

· 「캐릭터명 ↵ 대사」 서식, 연속 채팅 구분선 정리, 숨김 메시지 삭제는 자동 적용됩니다.

텍스트 스타일

화자 설정 (아바타 교체 · 나레이션 지정)

파싱하면 아바타 목록이 표시됩니다.

롤 테이블 CSS

· CoC 7판 / inSANe 룰의 롤템플릿 스타일이 기본 내장되어 있습니다.

내보내기

EPUB 폭·여백 커스텀 설정 (기본값 사용 시 열 필요 없음)
EPUB 이미지 압축 설정 (큰 이미지 축소)
· 최대 폭보다 작은 이미지는 손대지 않습니다. 큰 이미지만 가로 폭을 이 값으로 줄이고, 세로는 비율에 맞춰 자동 조정됩니다.
· 투명(알파)이 있는 PNG는 JPEG로 바꾸면 배경이 검게 되므로, 원본 형식 유지 또는 WebP를 권장합니다.
